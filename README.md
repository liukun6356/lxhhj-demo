- Node 环境
  版本：18.0.0
- 开发工具
  Webstorm

## 项目启动
1. 安装依赖

    ```bash
    yarn install
    ```
2. 启动运行

    ```bash
    yarn dev
    ```
3. 访问测试

   浏览器访问： [--](--)
4. 测试环境打包
    ```bash
    yarn build:staging
    ```
5. 正式环境打包
    ```bash
    yarn build:prod
    ```
